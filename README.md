# AWS Developer Associate — Study Plan

Interactive slide-based study materials for the **AWS Certified Developer Associate (DVA-C02)** exam.

## Structure

```
aws-study-plan/
├── index.html                    # Main page — all 20 weeks
├── week1-iam-aws-cli.html        # Week 1: IAM & AWS CLI
├── week2-ec2-fundamentals.html   # Week 2: EC2 Fundamentals
└── README.md
```

## How to Use

1. Put all HTML files in the same folder
2. Open `index.html` in any browser
3. Click on any available week to open the slides
4. Navigate slides with arrow keys, spacebar, or click the navigation dots
